---|---|
1주차|리소스 수집(100%)|

3주차|이동하는 몬스터 구현(100%)|
4주차|포탑 설치와 공격 구현(100%)|
5주차|몬스터와 포탑간의 충돌 구현(100%)|
6주차|상점 UI 구현(70%)|
7주차|인게임 UI 구현(50%)|

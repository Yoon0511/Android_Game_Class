# 타워디펜스
+ 게임컨셉
  + 타일 맵기반 타워 디펜스
  + 모든 몬스터를 막아내면 승리
  + 몬스터를 잡고 획득한 재화로 타워를 설치 및 강화
  + 일정 시간마다 몬스터가 몰려오는 BigWave가 존재
# 개발 범위
  + 타워
    + 드래그 앤 드롭으로 설치
    + 광역/슬로우 등 다양한 형태로 공격
    + 재화를 통한 강화 및 구매
  + 몬스터
    + 자동 이동
    + 재화 드랍
    + 체력/이동속도
  + 타일 맵
    + 몬스터가 이동하는 구역
    + 타워를 설치할 수 있는 구역
  + 상점
    + 재화를 소모해서 타워의 성능을 강화
